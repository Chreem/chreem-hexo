## syntax

### 箭头函数

()=>{} 约等于匿名函数 function(){}  
其他部分[@baidu](https://www.baidu.com)，此处主要是差别，主要是this指向略有不同：  

```js
//就把String拎出来示众了……
//function(){}
String.prototype.es5 = function(){ console.log(this); }
String.prototype.es6 =       ()=>{ console.log(this); }

//然后再把vue拉出来示众
new Vue({
    mounted(){
        let hehe = '仅供测试';
        hehe.es5();             //仅供测试
        hehe.es6();             //log出Vue Obj
    },
    data(){
        return {};
    }
});

/** 简之，function会改变this指向而箭头不会 */
```
